准备工作

安装 Anaconda

Linux/macOS 都内置了 Python，足够日常使用，但强烈建议不要使用系统内置的 Python，而建议安装 Anaconda，并使用 conda 管理 Python 环境。详情请参考 Anaconda。

安装 ObsPy

接下来的实践操作需要使用 ObsPy 包。可以通过如下命令安装 ObsPy:

$ conda install obspy
